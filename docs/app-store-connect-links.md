# App Store Connect Links

Use these URLs after deploying the static reviewer site to Vercel.

- Marketing URL: `https://strollmatesappi.vercel.app/`
- Privacy Policy URL: `https://strollmatesappi.vercel.app/privacy.html`
- User Privacy Choices URL: `https://strollmatesappi.vercel.app/privacy.html#choices`
- Support URL: `https://strollmatesappi.vercel.app/support.html`
- Terms and Conditions URL: `https://strollmatesappi.vercel.app/terms.html`

The in-app Settings screen uses the same Privacy Policy, Terms and Conditions, Support, purchase restore, subscription-management, and account-deletion paths.

Use `docs/app-store-metadata.md` for the App Store Connect name, subtitle, promotional text, description, keywords, review notes, and EULA setting.
